# ADE - Awesome Dance Effects

2014年 苫小牧高専 高専祭にて、ダンス部・美術同好会・情報処理同好会でコラボしてダンスパフォーマンスを行いました。
背景で流している映像は、openFrameworksで作成した映像をプロジェクターで投影したものです。
